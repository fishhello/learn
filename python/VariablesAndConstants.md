## 变量与常量

### 常量：永远不会改变值的对象
>没有自变量对象称为常量,下面示例中每一个数字都是常量

```python
1 + 2
2 * 2
```

### 变量：相反，并使用赋值符赋值

```java
//相比其他编程语言，如java声明一个变量为：
int a;
a=1;
System.out.println(a);
```

```python
#而python则更简洁
a=1
print(a)
```

### 改变变量值
>不需要声明变量的类型，直接赋值可以判断变量类型

```python
x = 1
x += 1
x -= 1
```
```python
my_float = 2.2
my_boolean = true
```
